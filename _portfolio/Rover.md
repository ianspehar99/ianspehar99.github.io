Developed an autonomous navigation adn search policy for the OSU Mars Rover, tested in a custom Gazebo Harmonic simulation environment. The goal was to prepare the rover for robot competitions that provide bonus points for efficient, autonomous search in an area filled with obstacles. 

![RC Car](/images/sim_sc.png)

Wrote all software and created the simulation for algorithm development and proof of concept.

<!-- <img src="/images/car2.png" alt="Another view of RC Car" style="max-width:400px;"> -->


Created an autonomous navigation mode for the rover, allowing it to navigate around obstacles and find a red flag without operator interference.
